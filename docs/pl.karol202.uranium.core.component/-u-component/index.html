UComponent

(common, js, jvm, wasm32) interface UComponent<N, P : UProps> : UPropsProvider<P>

Components are basic building blocks for all uranium applications, they form a tree structure, every component can have any number of children component or may have no children.

Every component has its own lifecycle:

|------------------|

| Instantiated |

|------------------|

     |   create() / onCreate()

    \/

|-------------|

| Created | render() and onUpdate() are called

|-------------|

     |   destroy() / onDestroy()

    \/

|---------------|

| Destroyed |

|---------------|
