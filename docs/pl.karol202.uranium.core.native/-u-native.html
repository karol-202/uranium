UNative

(common, js, jvm, wasm32) interface UNative<N>

Native is an object referring to platform's entity such as view, control or DOM node. Tree of natives is a final result of committing phase.

UNative is an interface, that all natives must implement. Every platform implementation has to have its own natives implementations.

Parameters

N - native marker

Inheritors

(common, js, jvm, wasm32)

UNativeContainer

UNativeContainer is a special UNative that itself can contain other natives (maybe native containers).
