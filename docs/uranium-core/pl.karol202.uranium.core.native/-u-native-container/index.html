UNativeContainer

(common, js, jvm, wasm32) interface UNativeContainer<N> : UNative<N>

UNativeContainer is a special UNative that itself can contain other natives (maybe native containers).

On some platforms all natives implement UNativeContainer (for example Swing, where JComponents is used as by native objects and all JComponents are containers), and on other platforms this is not the case.
